** This project demonstrate Angularjs 1.6 component based application.
** It is fully ES6 using webpack (babel).
** Based on Toddmodo styleguide.

** instructions:

	- clone the project
	- run npm i
	
** for dev needs:

	- run npm start

** for serving as a product: 

	- run npm run build
	- serve the 'dist' content
